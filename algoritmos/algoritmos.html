Algoritmo: Conversión de Base (Divisiones Sucesivas)

1. Marco Teórico

Normalmente usamos el sistema decimal (base 10), que usa 10 dígitos (0-9). Sin embargo, las computadoras usan el sistema binario (base 2), con solo dos dígitos (0 y 1). El sistema hexadecimal (base 16) se usa mucho en computación para abreviar números binarios largos, y utiliza 16 símbolos (0-9 y A-F, donde A=10, B=11, ..., F=15).

Este algoritmo nos permite convertir un número de nuestra base 10 a cualquier otra base (como 2 o 16).

2. Entrada y Salida

Input (Entrada):
Un número entero positivo expresado en base 10 (decimal).
Output (Salida):
La representación de ese mismo número en la base nueva (2 o 16). Se interpreta leyendo los dígitos de izquierda a derecha, multiplicados por la potencia correspondiente de la nueva base.

3. Método (Pseudocódigo)

Usamos el método de divisiones sucesivas:


FUNCIÓN convertir(numero, base):
  SI numero ES 0:
    DEVOLVER "0"
  
  lista_de_restos = []

  MIENTRAS numero > 0:
    cociente = numero DIVISIÓN_ENTERA base
    resto = numero MÓDULO base
    
    AÑADIR resto a lista_de_restos
    
    numero = cociente
  
  INVERTIR lista_de_restos
  UNIR lista_de_restos en un string
  DEVOLVER string_resultado

Algoritmo: Descomposición en Factores Primos

1. Marco Teórico

Un número primo es un número natural mayor que 1 que solo tiene dos divisores: 1 y él mismo (ej: 2, 3, 5, 7, 11...).

El Teorema Fundamental de la Aritmética (que vimos en el repartido, fuente: 3488) nos garantiza que todo número compuesto (mayor que 1 y no primo) puede escribirse de forma única como un producto de números primos.

Este algoritmo encuentra esa lista única de factores primos.

2. Entrada y Salida

Input (Entrada):
Un número entero mayor que 1.
Output (Salida):
La lista de números primos cuyo producto es igual al número de entrada. (Ej: para 60, la salida es "2 x 2 x 3 x 5").

3. Método (Pseudocódigo)

Usamos el método de divisiones de prueba (trial division):


FUNCIÓN factorizar(numero):
  lista_de_factores = []

  // Primero, sacamos todos los factores '2'
  MIENTRAS numero ES DIVISIBLE POR 2:
    AÑADIR "2" a lista_de_factores
    numero = numero / 2
  
  // Ahora, probamos con impares desde 3
  // Solo necesitamos probar hasta la raíz cuadrada de 'numero'
  divisor = 3
  MIENTRAS divisor * divisor <= numero:
    MIENTRAS numero ES DIVISIBLE POR divisor:
      AÑADIR divisor a lista_de_factores
      numero = numero / divisor
    divisor = divisor + 2
    
  // Si 'numero' es > 1 al final, es porque él mismo es primo
  SI numero > 1:
    AÑADIR numero a lista_de_factores
    
  DEVOLVER lista_de_factores
